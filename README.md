# 🏛️ EvenueHub – Event Venue Booking System

EvenueHub is a web-based event venue booking system that allows users to browse, book, and manage event venues. It includes user authentication, venue listings, booking management, and payment tracking.

---

## 🛠️ Tech Stack

- PHP (Core Backend)
- MySQL (Database)
- Apache (XAMPP Server)
- HTML, CSS, JavaScript
- phpMyAdmin

---

## ⚙️ Installation Guide

### 1. Install Required Software

- Install **XAMPP**: https://www.apachefriends.org/
- Install **Visual Studio Code**: https://code.visualstudio.com/

---

### 2. Start Local Server

1. Open XAMPP Control Panel  
2. Start:
   - Apache
   - MySQL

---

### 3. Project Setup

1. Copy the project folder:
evenuehub

2. Paste it into:
C:\xampp\htdocs\
---

### 4. Create Database

1. Open phpMyAdmin
2. Create a database named: evenuehubdb
3. Select the database evenuehubdb
4. Go to Import tab
5. Choose the file: evenuehubdb.sql
6. Click Import
---

### 5. Run the Project

Open in browser:
http://localhost/evenuehub/index.php
